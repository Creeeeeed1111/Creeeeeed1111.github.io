Vulkan 渲染器 (Vulkan Renderer)

C++ | Vulkan API | MoltenVK

这是一个基于 Vulkan API 实现的高性能图形渲染器。实现了 physically based shading (PBR) 模型，支持 Lambertian 漫反射和 microfacet BRDF 镜面反射。

View Source
技术实现细节：
  • **PBR 材质**：实现了基于 Beckmann 法线分布函数的微表面 BRDF。
  • **法线贴图 (Normal Mapping)**：增加了表面细节。
  • **Debug 模式**：支持 Mipmap 等级、深度缓存、顶点密度等多种可视化调试视图。
  • **跨平台**：利用 MoltenVK 在 macOS 上成功运行。
