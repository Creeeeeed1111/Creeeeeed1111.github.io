High-Performance Vulkan PBR Engine
Graphics
C++ Vulkan GLSL MoltenVK

A high-performance 3D graphics rendering engine built from scratch. Features a complete Physically Based Rendering (PBR) pipeline with a robust low-level architecture supporting two-pass dynamic rendering and complex queue scheduling.

  • • Physically Based Rendering: Implemented Cook-Torrance BRDF (Beckmann ND, Schlick's Fresnel) with full support for metalness/roughness/normal mapping.
  • • Two-Pass Architecture: Separated geometry passes from post-processing using offscreen rendering and Push Constants for efficient full-screen effects.
  • • Advanced Queue Management: Render queue sorting for Opaque (back-face culling) and Alpha-Cutout (double-sided) materials; rigorous sRGB/UNORM color space management.
  • • Debugging & Iteration: Automated Mipmap chain generation; visual debugging for Albedo, LOD levels, and Depth; supports Real-Time Shader Hot-Swapping.
  • • Cross-Platform: Smooth on both Windows and Linux.
